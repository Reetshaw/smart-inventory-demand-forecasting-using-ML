# smart-inventory-demand-forecasting-using-ML
Developed a machine learning-based system to predict product demand using historical sales data. Implemented data preprocessing, model training, and evaluation for accurate forecasting. The system helps optimize inventory management, reduce stock shortages, and improve business decision-making using Python, Pandas, NumPy, and Random Forest
# Smart Inventory & Demand Forecasting System

This project predicts product demand using Machine Learning and helps manage inventory efficiently.

## Features
- Demand Prediction
- Inventory Management
- Data Visualization
- Machine Learning Model

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## How to Run

pip install -r requirements.txt
streamlit run app.py
